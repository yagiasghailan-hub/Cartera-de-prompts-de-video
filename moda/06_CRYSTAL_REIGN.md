## Estilo (Iluminación / Paleta / Gusto Visual)
- **Iluminación**: 
  - Iluminación suave de estudio con reflectores direccionales
  - Luz clave desde arriba para crear destellos en cristales
  - Fill light suave para evitar sombras duras
- **Paleta de color**:
  - Primarios: Blanco puro, tonos piel cálidos
  - Secundarios: Destellos de arcoiris (prisma natural de los cristales)
  - Acentos: Plateado/gris perla para el hardware del collar
- **Gusto visual**: Clean luxury, editorial de revista de moda, estética de joyería fina

---

## Cámara (Movimiento / Lente / Velocidad / Técnicas)
- **Movimiento**: 
  - Push-in lento hacia el collar en el cuello
  - Ligera rotación orbital para capturar refracción de luz
  - Macro deslizante sobre las facetas de los cristales
- **Lente**: Macro 100mm para detalles, 50mm f/1.4 para contexto
- **Velocidad**: Movimientos lentos y fluidos (real-time)
- **Técnicas especiales**:
  - Rack focus entre cristal y piel
  - Shallow depth of field para bokeh suave
  - Reflejos especulares controlados

## Notas de Producción
- Usar cristales auténticos de alta calidad para refracción genuina
- Iluminación debe ser cuidadosamente controlada para evitar overexposure
- Modelo debe tener tono de piel que contraste bellamente con el collar
- Considerar fotografía de alta velocidad para capturar movimiento de refracciones
- Maquillaje y styling minimalista para no competir con la joyería
- Asegurar que cada cristal esté impecablemente limpio antes de filmar
